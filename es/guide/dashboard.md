# Dashboard

Personaliza tu pantalla de inicio con los widgets más relevantes para tu rutina de trabajo.

## Visión General

El Dashboard de Interflow es totalmente personalizable para cada usuario. Puedes elegir, reordenar y eliminar widgets de los módulos disponibles — CRM, financiero, citas, tareas y conversaciones — dejando la información más importante justo al entrar al sistema.

::: tip 💡 Acceso
Accede al Dashboard desde el **ícono de inicio** en el menú lateral.
:::

## Widgets Disponibles

| Widget | Descripción |
|--------|-------------|
| **Estadísticas Personales** | Tarjetas con resumen de conversaciones activas, tareas y citas |
| **Conversaciones Recientes** | Últimas conversaciones asignadas al usuario |
| **Tareas — Lista** | Tareas pendientes y en progreso |
| **Tareas — Kanban** | Vista completa en columnas por etapa |
| **Citas** | Próximas citas agendadas |
| **Embudo CRM** | Resumen de clientes por etapa del embudo de ventas |
| **Resumen Financiero** | Saldos de caja y visión financiera general |

## Personalizando el Dashboard

1. Accede al **Dashboard** desde el menú lateral
2. Haz clic en el botón **"Personalizar"** en la esquina superior derecha del banner de bienvenida
3. En el modal, **arrastra** los widgets para reordenar o usa los íconos para:
   - **Cambiar tamaño** (↔ entre media anchura y anchura completa)
   - **Eliminar** el widget (🗑)
4. En la sección **"Agregar widgets"**, haz clic en **+** para incluir nuevos módulos
5. Haz clic en **"Guardar configuración"** — el diseño queda guardado para futuras sesiones

::: info 💾 Sincronización
La configuración se guarda en la base de datos y se sincroniza entre todos tus dispositivos.
:::

## Visualizar como Usuario (Administradores)

Los administradores y propietarios pueden verificar cómo aparece el dashboard para usuarios normales:

1. En el dashboard admin, haz clic en **"Visualizar como usuario"**
2. Ve y configura el dashboard en modo usuario normal
3. El modo seleccionado se guarda y restaura automáticamente
