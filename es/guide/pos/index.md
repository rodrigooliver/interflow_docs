# PDV (Punto de Venta)

Ventas de mostrador, pedidos, productos, inventario y mesas.

::: tip Acceso
Menú → **PDV**. Requiere el módulo `pos`.
:::

## Visión general

| Área | Función |
|------|--------|
| **Panel** | KPIs y atajos |
| **Ventas** | Carrito, código de barras, pago |
| **Productos** | Catálogo, variantes, stock |
| **Pedidos** | Pedidos y presupuestos |
| **Caja** | Sesión de caja |
| **Inventario** | Niveles y movimientos |
| **Mesas** | Salón |
| **Configuración** | Tienda, recibo, pagos, mesas |

## Flujo típico de venta

1. Abra **PDV → Ventas**
2. Agregue productos (búsqueda o código de barras)
3. Opcional: vincule **cliente** y/o **mesa**
4. **Finalizar venta** → elija pago → confirme

## Tipos de pedido

PDV · Presupuesto · Entrega · Retiro · Consumo en el local
