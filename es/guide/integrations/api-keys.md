# Claves de API

Genere claves para la [API de Interflow](/es/api/).

::: tip Acceso
**Configuración → API Keys**
:::

1. **Nueva clave** → nombre → copie una vez → guarde segura
2. Autentíquese según [Autenticación](/es/api/authentication)

::: danger
La clave secreta suele mostrarse **una sola vez**.
:::
