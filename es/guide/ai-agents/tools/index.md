# Herramientas de IA

Configure las acciones que su Agente IA puede ejecutar durante las conversaciones.

## Tipos de Herramientas

### 1. Acciones Listas (Sistema)
Acciones preconfiguradas de Interflow, listas para usar. Solo agregue y personalice.

### 2. Acciones Personalizadas
Herramientas personalizadas que usted crea para integraciones específicas.

## Acciones listas

| Acción | Descripción |
|--------|-------------|
| [Cambiar Nombre del Cliente](/es/guide/ai-agents/tools/change-customer-name) | Actualiza el nombre del cliente en el registro durante la conversación |
| [Actualizar Datos del Cliente](/es/guide/ai-agents/tools/update-customer-data) | Guarda campos personalizados informados en el chat |
| [Cambiar Contacto del Cliente](/es/guide/ai-agents/tools/change-customer-contact) | Guarda o actualiza teléfono, WhatsApp o correo |
| [Cambiar Dirección del Cliente](/es/guide/ai-agents/tools/change-customer-address) | Guarda dirección (texto o GPS) en el registro del cliente |
| [Consultar API](/es/guide/ai-agents/tools/query-api) | Consulta o ejecuta operaciones en APIs externas con rutas permitidas |
| [Investigar información en sitios](/es/guide/ai-agents/tools/research-websites) | Lee páginas web (Firecrawl o HTML) para responder con contenido actualizado |
| [Transferir a Equipo](/es/guide/ai-agents/tools/transfer-to-team) | Deriva la atención a un equipo humano |
| [Agendar](/es/guide/ai-agents/tools/schedule) | Crea citas desde la conversación |
| [Confirmar Cita](/es/guide/ai-agents/tools/confirm-schedule) | Confirma asistencia en citas existentes |
| [Cambiar Embudo](/es/guide/ai-agents/tools/change-funnel) | Mueve al cliente entre etapas del embudo de ventas |
| [Cuando No Sepa Responder](/es/guide/ai-agents/tools/unknown-response) | Define qué hacer cuando la IA no encuentra la respuesta |
| [Cambiar Fecha de Retorno](/es/guide/ai-agents/tools/change-return-date) | Reprograma el follow-up automático del cliente |
