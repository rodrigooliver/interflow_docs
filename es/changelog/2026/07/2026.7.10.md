## ✨ Nuevas Funcionalidades

### Herramienta: Consultar API

Nueva acción lista del Agente de IA para **consultar y ejecutar operaciones en una API externa** durante la atención — con base URL, autenticación cifrada y lista de rutas permitidas.

#### ¿Para qué sirve?

- Consultar pedido, cobro, stock o registro en sistemas externos (Mercado Pago, ERP, etc.)
- Crear o actualizar registros vía API REST sin exponer la clave al modelo
- Liberar solo las rutas necesarias (allowlist), con seguridad en el servidor

#### Cómo usar

1. Abra el **Agente de IA** (Prompt) y vaya a **Acciones / Herramientas**
2. Agregue la acción **Consultar API**
3. Configure base URL, autenticación y rutas — o use **Generar con IA**
4. Pruebe las rutas que quiera y guarde el prompt

> Guía completa: [Herramientas de IA — Consultar API](/es/guide/ai-agents/tools#consultar-api)

### Generar rutas de API con IA

Asistente en el modal **Consultar API** que arma la configuración a partir de una descripción en lenguaje natural, enlace de documentación, fragmentos de la doc o ejemplos de `curl`.

#### ¿Para qué sirve?

- Acelerar la configuración sin montar cada ruta a mano
- Generar solo las rutas necesarias para lo que el agente debe hacer
- Completar automáticamente nombre, descripción y slug de la acción

#### Cómo usar

1. Al agregar **Consultar API** (acción nueva), el asistente se abre automáticamente
2. Describa lo que el agente necesita hacer y, si tiene, indique la URL de la documentación
3. Opcional: pegue fragmentos de la doc o ejemplos de llamadas (`curl`)
4. Haga clic en **Generar**, revise el preview y **Aplicar**
5. Ajuste y pruebe las rutas antes de publicar

> Guía completa: [Herramientas de IA — Consultar API](/es/guide/ai-agents/tools#consultar-api)

---

### Probar rutas de la API

Cada ruta permitida puede **probarse en el servidor** (la clave no se expone en el navegador), con estado de la última prueba (OK / Error / Sin probar).

#### ¿Para qué sirve?

- Validar autenticación y path antes de ir a producción
- Ver mensaje y HTTP status de la última prueba
- Evitar publicar rutas que aún fallan

#### Cómo usar

1. Abra la configuración de **Consultar API**
2. En cada ruta, use **Probar ruta** y complete parámetros de ejemplo
3. Revise el estado y ajuste si hace falta

> Guía completa: [Herramientas de IA — Consultar API](/es/guide/ai-agents/tools#consultar-api)

## 🔄 Mejoras

### Modal dedicado y nombre más claro

La configuración de API externa ahora tiene modal propio (**Configurar consulta en API**) y el nombre predeterminado de la acción pasó de “API Configurable” a **Consultar API**, alineado con las demás acciones listas.
