## ✨ Nuevas Funcionalidades

### 👥 Selección de Equipo al Iniciar una Conversación

Ahora es posible elegir el equipo responsable de una atención en el momento en que se crea, ya sea en conversaciones individuales, campañas de envío masivo o integraciones vía API.

#### ¿Para qué sirve?

- **Control en la distribución** – Define qué equipo recibe la atención desde el primer contacto, sin necesidad de transferencias posteriores
- **Conversaciones individuales** – Al iniciar una conversación con un cliente, aparece un selector de equipo antes de elegir el canal
- **Envíos masivos** – El campo "Equipo para chats creados" en la campaña asegura que todas las nuevas atenciones generadas vayan al equipo correcto
- **Vía API** – Las integraciones externas pueden enviar el campo `teamId` para definir el equipo automáticamente

#### ¿Cómo usar?

1. **Conversación individual:** Abre el perfil de un cliente → haz clic en **Iniciar conversación** → elige el equipo en el selector → selecciona el canal
2. **Envío masivo:** Crea o edita una campaña → busca el campo **"Equipo para chats creados"** → selecciona el equipo deseado
3. **API:** Envía el campo `teamId` en el cuerpo de la solicitud al crear una atención

::: info Asignación automática
Si no se selecciona ningún equipo, el sistema sigue el orden automático: **predeterminado del canal → equipo del usuario → equipo predeterminado de la organización**.
:::

## 🐛 Correcciones

### Un número inválido ya no crea una atención

Para canales de WhatsApp con validación de número activa, cuando el contacto no tiene WhatsApp, la atención **no se bloqueaba correctamente** y no se mostraba ningún error claro. Ahora el sistema bloquea la creación y muestra el mensaje de error exacto, facilitando la identificación del problema.

## 🎯 Beneficios

- ✅ **Distribución inmediata** – Las atenciones llegan directamente al equipo correcto, sin pasos manuales de transferencia
- ✅ **Consistencia en campañas** – Todos los chats generados por una campaña ya nacen con el equipo definido
- ✅ **Integración simplificada** – Las APIs y automatizaciones pueden controlar el equipo de destino de forma programática
- ✅ **Errores más claros** – Crear una atención para un número inválido ahora falla de forma clara e informativa
