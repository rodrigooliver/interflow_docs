## Nuevas Funcionalidades

### Orden Alfabético de Agentes IA

Ahora es posible organizar la lista de Agentes IA en orden alfabético.

#### ¿Cómo usar?

1. Accede a la lista de **Agentes IA**
2. Usa el campo de **búsqueda** para encontrar agentes por nombre
3. Haz clic en el botón **A → Z** para alternar el orden:
   - **A → Z** = orden alfabético (A primero)
   - **Z → A** = orden inverso (Z primero)

::: tip Consejo
¡La lista ya abre organizada de A a Z automáticamente!
:::

### Menú Financiero Restringido para Administradores

El menú financiero ahora está oculto para agentes, manteniendo acceso solo para perfiles administrativos.

#### ¿Quién tiene acceso al módulo Financiero?

| Rol | Acceso |
|-----|--------|
| `owner` | ✅ Acceso total |
| `admin` | ✅ Acceso total |
| `manager` | ✅ Solo lectura |
| `financial` | ✅ Acceso total |

#### ¿Quién YA NO tiene acceso?

| Rol | Descripción |
|-----|-------------|
| `agent` | Agente |
| `agent_limited` | Agente Limitado |
| `sales` | Ventas |
| `medical_doctor` | Médico |
| `medical_assistant` | Asistente Médico |
