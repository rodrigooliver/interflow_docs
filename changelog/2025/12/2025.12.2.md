## Novas Funcionalidades

### Ordenação Alfabética dos Agentes IA

Agora é possível organizar a lista de Agentes IA em ordem alfabética, facilitando a localização rápida do agente desejado.

#### Como usar?

1. Acesse a lista de **Agentes IA**
2. Use o campo de **busca** para encontrar agentes pelo nome
3. Clique no botão **A → Z** para alternar a ordem:
   - **A → Z** = ordem alfabética (A primeiro)
   - **Z → A** = ordem inversa (Z primeiro)

::: tip Dica
A lista já abre organizada de A a Z automaticamente!
:::

### Menu Financeiro Restrito para Administradores

O menu financeiro agora está oculto para atendentes, mantendo acesso apenas para perfis administrativos.

#### Quem tem acesso ao módulo Financeiro?

| Role | Descrição | Acesso |
|------|-----------|--------|
| `owner` | Dono | ✅ Acesso total |
| `admin` | Administrador | ✅ Acesso total |
| `manager` | Gerente | ✅ Somente leitura |
| `financial` | Financeiro | ✅ Acesso total |

#### Quem NÃO tem mais acesso?

| Role | Descrição |
|------|-----------|
| `agent` | Atendente |
| `agent_limited` | Atendente Limitado |
| `sales` | Vendas |
| `medical_doctor` | Médico |
| `medical_assistant` | Assistente Médico |
