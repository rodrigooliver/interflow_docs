## Novas Funcionalidades

### Busca de Chats por Número de Telefone

Agora é possível encontrar conversas digitando o número de telefone.

#### Como usar?

1. Clique no ícone de lupa 🔍 no cabeçalho
2. Selecione a aba **"Chats"**
3. Digite o número (completo ou parcial)
4. Pressione Enter

#### Exemplos de busca:
```
5511987654321    (número completo)
11987654321      (sem código do país)
4321             (últimos dígitos)
11987            (DDD + início)
```

#### Abas disponíveis:
| Aba | Para que serve |
|-----|----------------|
| Clientes | Encontrar clientes e iniciar conversa |
| **Chats** ⭐ | Encontrar conversas existentes (NOVO!) |
| Mensagens | Buscar conteúdo de mensagens |

### Configurar Aba Padrão ao Acessar

Escolha qual tela abre automaticamente ao fazer login.

#### Como configurar (para você):

1. Clique no seu **avatar** (canto inferior esquerdo)
2. Encontre **"Tela Inicial"**
3. Escolha o módulo desejado
4. Clique em **"Salvar"**
5. Faça login novamente

#### Módulos disponíveis:
- 🔵 Dashboard
- 💚 Conversas
- 👥 Clientes
- 💜 CRM
- 🟠 Agendamentos
- 🔴 Médico
- 🔷 Tarefas
- 💰 Financeiro
- 🟡 PDV
