## Novas Funcionalidades

### Conversão de JSON para Texto Formatado

Novo recurso no nó de **Alterar Variável** do fluxo para converter JSON em texto legível.

#### O que faz?
- Transforma dados JSON em texto formatado (Markdown)
- Apresenta informações técnicas de forma clara ao cliente
- Elimina códigos e símbolos difíceis de entender

#### Como usar?
1. No nó **Alterar Variável**, ative a opção de conversão
2. O sistema transforma automaticamente o JSON em texto organizado
3. Use o resultado em mensagens ao cliente

---

### Relatório de Atendimentos Avançado

Novo relatório com filtros completos para análise de atendimentos.

#### Filtros disponíveis:
- **Setor/Equipe** - Um ou mais setores
- **Usuário** - Atendentes específicos
- **Período** - Intervalo de datas (criação, última mensagem, conclusão)
- **Status** - Aberto, pendente, finalizado, etc.

#### Funcionalidades:
- Visualização em lista com contagens
- Ordenação e agrupamento de resultados
- Exportação para CSV
- Análise de performance por equipe

---

### Melhorias no Sistema de Implantações

Atualizações no gerenciamento de solicitações de implantação.

#### Novidades:
- **Horas de Mão de Obra** - Substitui "Tempo" para maior clareza
- **Edição de urgência** - Ajuste prioridade após criação
- **Correções de bug com créditos** - Bugs geram créditos positivos (compensação)
- **Upload de múltiplos arquivos** - Até 50MB por arquivo
