## Novas Funcionalidades

### Lembretes Automáticos 24/48h para Agendamentos

Sistema de notificações automáticas antes de agendamentos com confirmação via WhatsApp.

#### Como configurar?

1. Acesse **Agendamentos** → **Gerenciar** → aba **Alertas**
2. Clique em **+ Adicionar Template**
3. Configure:
   - **Tempo antes**: 24h, 48h, ou outros intervalos
   - **Canal**: Selecione o WhatsApp
   - **Serviços**: Escolha quais serviços recebem o lembrete
   - **Mensagem**: Personalize com variáveis (`{{name}}`, `{{date}}`, `{{hour}}`)
   - **Flow**: Vincule um flow de confirmação (opcional)

#### Confirmação automática:
- Cliente responde "SIM", "CONFIRMADO", etc.
- Sistema atualiza status automaticamente
- Registro no histórico do cliente

### Configuração Rápida Restrita para Admins

A seção de **Configuração Rápida** do painel agora é exclusiva para administradores.

#### O que muda?

| Role | Acesso |
|------|--------|
| Owner | ✅ Acesso total |
| Admin | ✅ Acesso total |
| Atendente | ❌ Não visualiza |

#### Benefícios:
- Maior segurança sobre configurações
- Reduz riscos de alterações indevidas
- Interface mais focada para atendentes
