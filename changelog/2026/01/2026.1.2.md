## Novas Funcionalidades

### Controle de Acesso Granular ao CRM

Implementado sistema completo de permissões que permite controlar exatamente o que cada atendente pode visualizar no módulo de CRM. Agora é possível garantir que cada usuário tenha acesso apenas aos dados dos clientes que atende diretamente.

#### Para que serve?

- ✅ **Privacidade e Segurança**: Dados confidenciais dos clientes ficam protegidos e acessíveis apenas aos responsáveis diretos
- ✅ **Conformidade LGPD**: Atende requisitos de proteção de dados e minimização de acesso a informações pessoais
- ✅ **Experiência Focada**: Atendente visualiza apenas o que precisa para seu trabalho, com interface mais limpa e objetiva
- ✅ **Controle Administrativo**: Gestores decidem o nível de acesso de cada usuário conforme necessidade do negócio

#### Como funciona?

**Para Atendentes Limitados (agent_limited)**:
1. No Menu CRM: Visualiza apenas a opção "Clientes que atendo"
2. Na Tela do Funil: Vê somente os clientes atribuídos a ele
3. Filtro Automático: O sistema automaticamente filtra apenas seus clientes
4. Sem Toggle: Botão de alternar entre "todos" e "meus clientes" fica oculto

**Para Outros Atendentes (agent)**:
1. No Menu CRM: Podem escolher entre "Clientes que atendo" e "Todos os clientes"
2. Na Tela do Funil: Podem usar um toggle para alternar a visualização
3. Permissões: Podem ler e editar (mas não criar ou excluir)

**Para Administradores e Gestores**:
- Acesso total sem restrições
- Visualizam todos os clientes
- Podem gerenciar pipelines e exportar dados

#### Como configurar?

1. Acesse o menu **Usuários** (área administrativa)
2. Selecione o usuário que deseja configurar
3. No campo **Permissão/Função**, escolha **"Atendente Limitado"** (agent_limited)
4. Salve as alterações

**Pronto!** O atendente só verá seus próprios clientes no CRM.

#### Tipos de Permissões Disponíveis

- **Owner/Admin**: Acesso total sem restrições
- **Manager**: Visualiza todos mas com algumas restrições administrativas
- **Agent (Atendente Normal)**: Acessa seus clientes + pode ver todos se necessário
- **Agent Limited (Atendente Limitado)**: ⭐ **Recomendado** - Vê APENAS seus clientes

### Consulta de Clientes por Permissão de Usuário

Agora a listagem de clientes respeita as permissões do usuário logado em todos os módulos do sistema (Clientes, CRM e Atendimentos).

#### Para que serve?

- ✅ **Segurança aprimorada**: Agentes com permissão `agent_limited` visualizam apenas clientes relacionados aos seus próprios atendimentos
- ✅ **Privacidade de dados**: Impede que agentes acessem informações de clientes não atribuídos a eles
- ✅ **Conformidade**: Atende requisitos de proteção de dados e privacidade
- ✅ **Controle granular**: Administradores continuam com acesso completo à base de clientes

#### Como funciona?

1. **Para Administradores e Supervisores**: Visualizam todos os clientes da equipe normalmente
2. **Para Agentes com `agent_limited`**: 
   - Visualizam apenas clientes dos atendimentos que foram atribuídos a eles
   - A listagem é automaticamente filtrada com base no campo `last_assigned_to`
   - Não há necessidade de configuração adicional

## Segurança

### Controle de Acesso Multinível a Dados de Clientes

- **Interface (UI)**: Oculta opções e controles não permitidos ao perfil do usuário
- **Backend**: Funções específicas no banco de dados (`get_customers_by_stage_v2`, `get_stage_customer_counts_v2`) filtram automaticamente os dados
- **Permissões**: Sistema de roles garante que apenas dados autorizados sejam acessados
- **Auditoria**: Histórico de acessos mantido para rastreabilidade

### Melhorias Técnicas

- Criadas versões v2 das funções de consulta do CRM com filtro por `last_assigned_to`
- Implementado hook `usePermissions` para controle de acesso baseado em roles
- Adicionadas constantes de permissões (`PERMISSION_TEMPLATES`) para todos os perfis
- Integração completa entre Sidebar, CRMFunnel e hooks de dados
