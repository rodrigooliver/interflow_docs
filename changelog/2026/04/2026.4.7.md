## ✨ Novas Funcionalidades

### Notificações de Agendamento para Profissionais

Os lembretes de agendamento agora podem ser enviados também para o **profissional responsável pelo atendimento**, além do cliente — ou exclusivamente para ele. Cada template de notificação passa a ter um campo **Destinatário** com três opções: apenas o cliente, apenas o profissional ou ambos.

#### Para que serve?

- **Lembrar o profissional** do próximo atendimento sem depender de processos externos.
- **Reduzir faltas e atrasos** por parte da equipe ao automatizar avisos via WhatsApp ou e-mail.
- **Flexibilidade total**: configure templates diferentes para o cliente e para o profissional com textos e horários de envio distintos.

#### Como funciona?

Ao criar ou editar um template de notificação de agendamento, escolha o destinatário:

| Opção | Comportamento |
|-------|---------------|
| **Cliente** | Envia apenas para o cliente do agendamento (comportamento anterior) |
| **Profissional** | Envia apenas para o profissional vinculado ao agendamento |
| **Ambos** | Envia para o cliente e para o profissional no mesmo disparo |

#### Sincronização automática de contato

O sistema vincula automaticamente um cadastro de cliente de notificação a cada membro da organização com base nos dados do perfil (WhatsApp e e-mail). A cada execução do envio de lembretes, o sistema:

1. Verifica se o membro já possui um cadastro de notificação vinculado.
2. **Se não tiver**: cria o cadastro automaticamente a partir do nome, WhatsApp e e-mail do perfil.
3. **Se o número de WhatsApp tiver mudado**: atualiza o contato existente para o número atual, garantindo que o profissional sempre receba a mensagem no número correto.
4. **Se um novo tipo de contato (e-mail) for adicionado ao perfil**: cria o contato complementar no cadastro.

Não é necessária nenhuma ação manual por parte do administrador — o sistema mantém os dados de contato sempre sincronizados.

## 🎯 Benefícios

- ✅ Profissionais recebem lembretes automáticos dos seus atendimentos via WhatsApp ou e-mail
- ✅ Configuração granular por template — diferentes textos e horários para cliente e profissional
- ✅ Cadastro de notificação criado automaticamente na primeira execução, sem configuração manual
- ✅ Contato sincronizado automaticamente caso o profissional troque de número
- ✅ Sem impacto nos fluxos existentes — templates sem destinatário definido continuam funcionando normalmente para o cliente
