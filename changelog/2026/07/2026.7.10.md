## ✨ Novas Funcionalidades

### Ferramenta: Consultar API

Nova ação pronta do Agente de IA para **consultar e executar operações em uma API externa** durante o atendimento — com base URL, autenticação criptografada e lista de rotas permitidas.

#### Para que serve?

- Consultar pedido, cobrança, estoque ou cadastro em sistemas externos (Asaas, ERP, etc.)
- Criar ou atualizar registros via API REST, sem expor a chave ao modelo
- Liberar só as rotas necessárias (allowlist), com segurança no servidor

#### Como usar?

1. Abra o **Agente de IA** (Prompt) e vá em **Ações / Ferramentas**
2. Adicione a ação **Consultar API**
3. Configure base URL, autenticação e rotas — ou use **Gerar com IA**
4. Teste as rotas que quiser e salve o prompt

> Guia completo: [Ferramentas da IA — Consultar API](/guide/ai-agents/tools#consultar-api)

### Gerar rotas de API com IA

Assistente no modal **Consultar API** que monta a configuração a partir de uma descrição em linguagem natural, link da documentação, trechos da doc ou exemplos de `curl`.

#### Para que serve?

- Acelerar a configuração sem montar cada rota na mão
- Gerar só as rotas necessárias para o que o agente precisa fazer
- Preencher automaticamente nome, descrição e slug da ação

#### Como usar?

1. Ao adicionar **Consultar API** (ação nova), o assistente abre automaticamente
2. Descreva o que o agente precisa fazer e, se tiver, informe a URL da documentação
3. Opcional: cole trechos da doc ou exemplos de chamadas (`curl`)
4. Clique em **Gerar**, revise o preview e **Aplicar**
5. Ajuste e teste as rotas antes de publicar

> Guia completo: [Ferramentas da IA — Consultar API](/guide/ai-agents/tools#consultar-api)

---

### Testar rotas da API

Cada rota permitida pode ser **testada no servidor** (a chave não é exposta no navegador), com status do último teste (OK / Erro / Não testado).

#### Para que serve?

- Validar autenticação e path antes de ir para produção
- Ver mensagem e HTTP status do último teste
- Evitar publicar rotas que ainda não funcionam

#### Como usar?

1. Abra a configuração de **Consultar API**
2. Em cada rota, use **Testar rota** e preencha os parâmetros de exemplo
3. Confira o status e ajuste se necessário

> Guia completo: [Ferramentas da IA — Consultar API](/guide/ai-agents/tools#consultar-api)
