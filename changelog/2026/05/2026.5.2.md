## ✨ Novas Funcionalidades

### Dashboard Personalizável

O dashboard inicial do sistema agora pode ser totalmente personalizado por cada usuário. É possível escolher, reordenar e remover widgets dos módulos disponíveis do Interflow — CRM, financeiro, agendamentos, tarefas e conversas — deixando as informações mais relevantes logo na entrada do sistema.

#### Para que serve?

- Montar uma visão inicial adaptada à sua rotina de trabalho
- Exibir os módulos que você mais utiliza sem precisar navegar por menus
- Reordenar os widgets arrastando e soltando conforme preferência
- Alternar entre visualização em meia largura ou largura total para cada widget
- Ter a configuração sincronizada entre dispositivos (salva no banco de dados)

#### Widgets disponíveis

| Widget | Descrição |
|--------|-----------|
| **Estatísticas Pessoais** | Cards com resumo de conversas ativas, tarefas e agendamentos |
| **Conversas Recentes** | Últimas conversas atribuídas ao usuário |
| **Tarefas — Lista** | Tarefas pendentes e em andamento |
| **Tarefas — Kanban** | Visualização completa em colunas por estágio |
| **Agendamentos** | Próximos compromissos agendados |
| **Funil CRM** | Resumo de clientes por etapa do funil de vendas |
| **Resumo Financeiro** | Saldo dos caixas e visão financeira geral |

#### Como usar?

1. Acesse o **Dashboard** pelo menu lateral
2. Clique no botão **Personalizar** no canto superior direito do banner de boas-vindas
3. No modal, **arraste** os widgets para reordenar ou clique nos ícones para:
   - **Alterar tamanho** (↔ entre meia largura e largura total)
   - **Remover** o widget (🗑)
4. Na seção **Adicionar widgets**, clique em **+** para incluir novos módulos
5. Clique em **Salvar configuração** — o layout fica salvo para próximas sessões

#### Visualização como usuário (Administradores)

Administradores e proprietários agora têm um botão **"Visualizar como usuário"** no dashboard admin. Ao ativá-lo, é possível ver e configurar o dashboard no modo de usuário normal. O modo selecionado é salvo e restaurado automaticamente a cada acesso.
