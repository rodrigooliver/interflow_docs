## New Features

### "My Tasks" Page

New page that gathers all tasks assigned to you, **regardless of project**, with list and calendar views.

#### What is it for?

- See all your tasks in one place, without navigating project by project
- Quickly identify which project each task belongs to — the project name is displayed next to the title
- Group your tasks by project, due date, status or priority

#### How to use it?

1. Go to **Tasks** and click the **"All my tasks"** button in the top bar
2. Switch between **List** and **Calendar** views
3. Use the search, overdue, archived, priority and status filters
4. Click **"View by project"** to return to the traditional view
5. Use the **"Refresh"** button to reload tasks instantly

### Grouping in the List View

The list view now lets you choose the grouping type, with the choice saved per organization.

#### What is it for?

- **Stage** (default) — traditional grouping by the project flow
- **Project** — available in "My Tasks", groups by the source project
- **Due date** — smart groups: Overdue, Yesterday, Today, Tomorrow, This week, Next week, Future and No date
- **Assignee** — one group per team member (in the project view)
- **Status** — Pending, In progress, Completed, Cancelled
- **Priority** — High, Medium, Low

### Sorting in the List View

Alongside grouping, you can now choose how tasks are sorted within each group:

- Default (priority + due date), closest/furthest due date, priority, title (A–Z / Z–A), creation date and status

### Quick Edit Directly in the List

Status, priority and due date can now be changed directly in the list, without opening the task.

#### How to use it?

1. In the list view, click on the **status**, **priority** or **due date** of any task
2. Pick the new value from the menu — for due dates there are shortcuts for **Today**, **Tomorrow**, **In 1 week**, a free date picker and remove date
3. Status changes (start, complete, cancel) **notify project members** and log the history, exactly like the existing actions

## Changes

### Visual Improvements in the Task List

- Date column header simplified to **"Due date"**, without line wrapping
- "In progress" status and dates no longer wrap
- Overdue tasks indicated by the red date only, without an extra icon
- Grouping and sorting controls in their own bar above the table
