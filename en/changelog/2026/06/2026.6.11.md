## New Features

### Push reminders before appointments (app and web)

The team now receives automatic alerts in the **mobile app** and **browser** before each appointment — configurable per schedule.

#### What is it for?

- Notify the provider and selected colleagues before the appointment time
- Reduce missed visits, consultations and meetings
- Complement WhatsApp/email reminders sent to the **customer** (separate channel)

#### How to configure?

1. Go to **Schedule** → select a schedule → **Edit** (or create a new schedule)
2. In **Notified users**, choose who besides the provider should receive alerts
3. Enable **Send push reminders before the appointment**
4. Select times: **15 min**, **30 min**, **1 h** or **2 h** chips
5. Optionally click **+ Add custom time** to set extra minutes or hours (up to 5 times total)
6. Save the schedule

> Full guide: [Schedule — Push reminders](/en/guide/schedule/push-reminders)

#### Behavior

| Aspect | Detail |
|--------|--------|
| **Default** | Enabled with **1 hour** reminder |
| **Recipients** | Appointment provider + users in **Notified users** |
| **Language** | Message based on each recipient's profile language |
| **Reschedule** | Reminder times recalculated automatically |
| **Cancellation** | Pending reminders are canceled |

## Changes

### Push on appointment creation

The push notification sent when an appointment is **created** was unified into the same reminders service, with translated messages (PT, EN, ES) based on each recipient's language.

## Benefits

- ✅ Internal reminders in app and web without relying on WhatsApp
- ✅ Per-schedule configuration with flexible times (presets + custom)
- ✅ Automatic sync on create, update or cancel appointments
- ✅ Messages in each team member's language
- ✅ Clear separation: push for team vs. templates for the customer
