## New Features

### Customizable Dashboard

The system's home dashboard can now be fully customized by each user. You can choose, reorder, and remove widgets from available Interflow modules — CRM, financial, appointments, tasks, and conversations — keeping the most relevant information front and center when you log in.

#### What is it for?

- Build a home view tailored to your daily workflow
- Display the modules you use most without navigating through menus
- Reorder widgets by dragging and dropping
- Toggle between half-width and full-width layout for each widget
- Configuration synced across devices (saved to the database)

#### Available widgets

| Widget | Description |
|--------|-------------|
| **Personal Statistics** | Summary cards for active chats, tasks, and appointments |
| **Recent Chats** | Latest conversations assigned to the user |
| **Tasks — List** | Pending and in-progress tasks |
| **Tasks — Kanban** | Full column view by stage |
| **Appointments** | Upcoming scheduled appointments |
| **CRM Funnel** | Customer summary by sales funnel stage |
| **Financial Summary** | Cash register balances and financial overview |

#### How to use?

1. Open the **Dashboard** from the sidebar
2. Click the **Customize** button in the top-right corner of the welcome banner
3. In the modal, **drag** widgets to reorder or use the icons to:
   - **Resize** (↔ toggle half-width / full-width)
   - **Remove** the widget (🗑)
4. Under **Add widgets**, click **+** to include new modules
5. Click **Save configuration** — the layout is saved for future sessions

#### View as User (Administrators)

Administrators and owners now have a **"View as user"** button on the admin dashboard. When activated, they can see and configure the dashboard in normal user mode. The selected mode is saved and automatically restored on each login.
