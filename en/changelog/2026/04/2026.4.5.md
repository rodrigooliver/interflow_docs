## ✨ New Features

### Attach files to customer notes

You can now include files directly in customer profile notes — photos, documents, PDFs, videos, audio and more — without needing external links.

#### What is it for?

- **Store important documents**: attach receipts, contracts, photos or any file relevant to the customer's history.
- **Centralize visual information**: keep images and documents in the same place as the note, without searching in another system.
- **Organized by note**: each note has its own file list, making it easy to find what was uploaded and when.

#### How to use?

**Attaching a file:**
1. Open or create a customer note (the **Notes** tab on the customer profile).
2. Click the **"Attach file"** button (paperclip icon) in the editor toolbar.
3. Select the file from your device.
4. The file is uploaded and automatically inserted into the note text:
   - **Images** appear visually inside the note body.
   - **Other files** (PDF, video, etc.) appear as a clickable link.

> If the note does not yet have a title, the system will ask you to provide one before proceeding with the upload.

**Viewing and managing note files:**

You can access the files in two ways:

- **From the notes list**: click the paperclip icon (purple) next to the delete button on the note card.
- **Inside an open note**: click the **"Files"** button in the top bar.

In the files screen you can:
- **View** all files attached to the note in a grid layout.
- **Open** any file in a new tab.
- **Insert** the file into the note text at the cursor position (available while editing).
- **Delete** a file individually — permanently removed from the system.

> When a **note is deleted**, all files linked to it are automatically removed as well.
