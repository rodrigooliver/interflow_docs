## ✨ New Features

### Message Reactions

Now you can react to messages with emojis directly in the chat, just like WhatsApp! Reactions make communication more expressive and dynamic.

#### What is it for?

- Quickly respond to messages without typing
- Express feelings and reactions visually
- Facilitate asynchronous communication between team and customers
- Create a more humanized service environment

#### How to use?

1. **Hover** over any message in the chat
2. **Click the emoji icon** (😊) that appears next to the message
3. **Select the desired emoji** from the picker that opens
4. The reaction will be sent instantly to the chat

#### Features

- **WhatsApp-like design**: Icon appears on hover
- **Smart positioning**: Button always appears on the opposite side of the message
- **Multiple reactions**: Several people can react to the same message
- **Compact design**: Overlapping reactions save space
- **Reaction toggle**: Click the same emoji again to remove your reaction
