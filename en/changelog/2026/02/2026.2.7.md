## ✨ New Features

### 📄 Document Templates – Complete System

Document generation system created from scratch, allowing you to create custom templates in **PDF** and **DOCX** with dynamic customer data, custom variables and conditional logic.

#### What's included?

**Visual Editor**
- HTML WYSIWYG editor with Jodit Editor
- Rich formatting (bold, italic, colors, tables, lists)
- Side panel with available variables to copy

**Dynamic Variables**
- **Customer Data**: Name, email, phone, address, documents
- **System Data**: Current date, organization, user
- **Custom Variables**: Model-specific fields (text, number, date, etc.)

**Handlebars Helpers**
- **Text**: <code v-pre>uppercase</code>, <code v-pre>lowercase</code>, <code v-pre>capitalize</code>, <code v-pre>titleCase</code>
- **Numbers in words**: PT, EN, ES (general and with currency)
- **Currencies**: <code v-pre>formatCurrencyBRL</code>, <code v-pre>formatCurrencyUSD</code>, etc.
- **Dates**: <code v-pre>formatDatePT</code>, <code v-pre>formatDateShortPT</code>, <code v-pre>formatDateTimePT</code>, day/month/year separate, GMT support
- **Mathematical operations**: <code v-pre>divide</code>, <code v-pre>multiply</code>, <code v-pre>add</code>, <code v-pre>subtract</code>, <code v-pre>sum</code>

**Conditionals**
- <code v-pre>#if</code>, <code v-pre>#ifExists</code>, <code v-pre>#ifNotEmpty</code>
- <code v-pre>#ifEquals</code>, <code v-pre>#ifGreaterThan</code>, <code v-pre>#ifLessThan</code>

**Export**
- PDF with custom margins and watermark
- Native DOCX with watermark
- Automatic download after generation

#### Use cases

- Contracts and commercial proposals
- Registration forms
- Reports with automatic calculations
- Multilingual documents (PT, EN, ES)

### 👥 Collaborators – Permission Control

Only **administrators** and **organization owners** can add, remove or manage collaborators in a conversation. **Limited agents** can only see who the collaborators are, without being able to change the list.
