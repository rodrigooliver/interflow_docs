## ✨ New Features

### Message Shortcut Sequences

You can now create **shortcuts with multiple messages** sent in order, with a **configurable interval** between each one — ideal for step-by-step greetings, sending material in parts, or follow-ups without typing message by message.

#### What is it for?

- Send **multiple messages in a row** with a single shortcut (`/`)
- Set the **interval between each message** (from 0.5s to 30s) to simulate a natural conversation pace
- Include **text and attachments** in each step of the sequence
- Control **agent signature** per message
- Keep **existing shortcuts** working as before (one step = previous behavior)

#### How to configure?

1. Go to **Organization** → **Message Shortcuts**
2. Create or edit a shortcut and use the **Messages** section to build the sequence
3. In each step, set the content, attachments, and whether the message should be signed
4. In each step footer (except the last), choose the **interval before the next message**
5. Use **Add message** for new steps (up to 10 messages per shortcut)
6. Save — in the list, shortcuts with more than one message show a badge with the count

> Full guide: [Message Shortcut Sequences](/en/guide/chat/message-shortcuts)

### Sequence Composer in Chat

When using a sequential shortcut in chat, a **review modal** opens so you can check and adjust everything before sending — without pushing the message history.

#### What is it for?

- **Review and edit** the text of each message before sending
- **Remove attachments** from specific steps without changing the saved shortcut
- **Adjust intervals** between messages at conversation time
- Send the entire sequence with **one click** (or **Enter** in the modal)
- See messages appear in chat **optimistically** while the backend processes the delays

#### How to use?

1. In chat, type `/` and select a shortcut with a multi-message badge
2. The modal opens with all steps — edit texts, attachments, and intervals as needed
3. Click **Send Sequence** or press **Enter**
4. To cancel, use **Cancel sequence**, **ESC**, or click outside the modal

> 💡 The normal message field **stays visible** while editing the sequence, so you don't lose conversation context.

> Full guide: [Message Shortcut Sequences](/en/guide/chat/message-shortcuts)
