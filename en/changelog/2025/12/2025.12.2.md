## New Features

### Alphabetical AI Agents Sorting

You can now organize the AI Agents list in alphabetical order.

#### How to use?

1. Access the **AI Agents** list
2. Use the **search** field to find agents by name
3. Click the **A → Z** button to toggle order:
   - **A → Z** = alphabetical order (A first)
   - **Z → A** = reverse order (Z first)

::: tip Tip
The list already opens organized A to Z automatically!
:::

---

### Financial Menu Restricted to Administrators

The financial menu is now hidden from agents, keeping access only for administrative profiles.

#### Who has access to the Financial module?

| Role | Access |
|------|--------|
| `owner` | ✅ Full access |
| `admin` | ✅ Full access |
| `manager` | ✅ Read-only |
| `financial` | ✅ Full access |

#### Who NO longer has access?

| Role | Description |
|------|-------------|
| `agent` | Agent |
| `agent_limited` | Limited Agent |
| `sales` | Sales |
| `medical_doctor` | Doctor |
| `medical_assistant` | Medical Assistant |
