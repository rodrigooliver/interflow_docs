## New Features

### Registration Field Indicator

Visual indicator in chat showing which fields are filled and missing.

#### How does it work?

- 📋 Panel appears next to the chat
- ✅ Green icon = Filled field
- ⚠️ Yellow icon = Missing field
- 🔴 Red icon = Required and missing

#### Benefit:
- Quickly see what data is missing
- Fill during conversation
- Click to go directly to field

---

### Auto Call Rejection for WhatsApp

Automatically rejects WhatsApp voice calls with configurable message.

#### How to configure?

1. Go to **Settings** → **Channels**
2. Select the WhatsApp channel
3. Locate **"Auto Call Rejection"**
4. Enable the option
5. Configure rejection message (optional)
6. Save

#### Example message:
> "Sorry, we don't receive calls. Please send a text message and we'll respond shortly!"

#### Behavior:
- 📵 Call is automatically rejected
- 💬 Configured message is sent
- 📊 Event logged for reports
- ✅ Works for audio and video calls
