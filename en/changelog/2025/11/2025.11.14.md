## New Features

### Recurring Appointments

Create recurring appointments for customers with regular schedules.

#### How to use?

1. When creating an appointment, enable **"Recurring"**
2. Configure recurrence:
   - **Frequency**: Daily, Weekly, Biweekly, Monthly
   - **End**: After X occurrences or specific date
   - **Days of week** (for weekly)
3. Click **"Create"**
4. All appointments are created at once

#### Management:

- ✏️ Edit single: Changes only selected occurrence
- ✏️ Edit all: Changes all future occurrences
- 🗑️ Delete single: Removes only one
- 🗑️ Delete all: Removes entire series

### Schedule Usability Improvements

Various improvements for better schedule experience.

#### What improved?

- 📅 **Expanded week view**: See up to 7 days at once
- 🖱️ **Drag and drop**: Move appointments between times
- 🎨 **Status colors**: Visual identification by status
- ⌨️ **Keyboard shortcuts**: Navigate faster
- 🔍 **Smart zoom**: Auto-adjust by period

#### Keyboard shortcuts:

| Shortcut | Action |
|----------|--------|
| `←` `→` | Navigate days |
| `T` | Go to today |
| `N` | New appointment |
| `Esc` | Close modal |
