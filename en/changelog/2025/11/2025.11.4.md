## New Features

### Restricted Customer Visibility for Agents

Agents now see only customers from their assigned teams.

#### How does it work?

- Agent accesses **Customers** list
- Only customers from teams they are part of appear
- Search also respects this restriction
- Admins continue seeing everyone

#### Affected areas:
- 📋 Customer list
- 🔍 Customer search
- 📊 Reports (filtered data)
- 📤 Exports (respects visibility)

### Custom AI Commands (Ctrl+J)

Create custom shortcuts to trigger AI commands in the chat input.

#### How to configure?

1. Go to **Settings** → **AI Commands**
2. Click **"New Command"**
3. Configure:
   - **Trigger**: Shortcut that activates (e.g.: `/summarize`)
   - **Prompt**: AI instruction
   - **Position**: Before or after cursor
4. Save

#### How to use?

1. In the chat input, press `Ctrl+J` or type `/`
2. Select the desired command
3. AI executes and inserts the result

#### Examples:
- `/summarize` - Summarizes conversation
- `/translate` - Translates to English
- `/tone` - Adjusts formal tone

---

### AI Message Signatures

Add automatic signature at the end of AI-generated messages.

#### How to configure?

1. Go to **AI Agent** → Edit agent
2. In **Settings**, locate **"Signature"**
3. Write the signature text
4. Save
