## New Features

### Pinned Messages in Chat

Highlight important information by pinning messages at the top of the chat.

#### How to pin?

1. Hover over the message
2. Click the three dots (⋮)
3. Select **"Pin message"**

#### Features:
- 📌 Messages appear at the top of the chat
- 🔢 Pinned message counter
- 🟡 Yellow background for pinned private notes
- ➜ "Go to message" button (navigates to original)
- 🔄 Real-time updates

#### How to unpin?
- Click **X** in the pinned list, or
- Message menu → **"Unpin message"**

---

### Private Note Node in Flow

New node to insert private messages automatically in the flow.

#### How to use?

1. In the flow editor, search for **"Private Note"** in **Actions**
2. Drag to the canvas
3. Configure the content (accepts variables)
4. Optional: Check **"Pin message to chat"**

#### Use example:
```
⚠️ Customer requested special discount - awaiting approval
```

---

### Option to Skip Flow on Transfer

When transferring a chat via AI, choose whether the destination team's flow should be executed.

#### How to configure?

1. Go to **AI Agents** → Edit agent
2. Go to **Tools**
3. Edit the **"Transfer to team"** action
4. In **Options**, check: ☑️ **"Do not execute related flow on transfer"**

#### Behavior:
- **Unchecked**: Destination team's flow is executed
- **Checked**: Only transfers, does not execute flow

---

### Reactivate Archived Conversation on New Message

Archived conversations automatically return to the queue upon receiving a new message.

#### How does it work?

- Agent archives a conversation
- Customer sends new message
- Conversation returns to main queue automatically
