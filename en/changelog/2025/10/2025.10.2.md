## New Features

### Cascade Update in Select Fields

When renaming a custom field option, all records are automatically updated.

#### How does it work?

1. Access a customer → **Custom Fields**
2. Click ✏️ next to the field
3. Click ✏️ next to the option you want to rename
4. Type the new value
5. Click **"Update"**

#### What gets updated:
- ✅ Field definition
- ✅ All customers using that option
- ✅ Shows how many records were affected

#### Protections:
- 🔒 Detects concurrent edits
- 🚫 Blocks duplicates
- ⚠️ Warns about conflicts

---

### Import Calendar Events with Date Filter

Import events from Google Calendar filtering by specific period.

#### How to use?

1. Export from Google Calendar (`.ics` file)
2. In **Appointments** → **New** → **Import Events**
3. Select the file
4. Configure date filter (optional)
5. Select schedule and professional
6. Click **Process File**
7. Map services and status for each event
8. Click **Import Events**

#### Available filters:
- ✅ Custom period (start and end date)
- ✅ Cancelled events are automatically ignored
