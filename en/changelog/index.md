## Version Format

We use the **YYYY.MM.N** format to identify versions:

| Component | Description |
|-----------|-------------|
| YYYY | Year |
| MM | Month |
| N | Sequential release number in the month (starts at 1) |

**Example:** `2025.12.3` = Third release of December 2025
