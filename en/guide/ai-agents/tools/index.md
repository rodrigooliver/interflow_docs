# AI Tools

Configure the actions your AI Agent can run during conversations.

## Tool Types

### 1. Ready-made Actions (System)
Preconfigured Interflow actions, ready to use. Just add and customize.

### 2. Custom Actions
Custom tools you create for specific integrations.

## Ready-made actions

| Action | Description |
|--------|-------------|
| [Change Customer Name](/en/guide/ai-agents/tools/change-customer-name) | Updates the customer name on the record during the conversation |
| [Update Customer Data](/en/guide/ai-agents/tools/update-customer-data) | Writes custom fields from what the customer shares in chat |
| [Update Customer Contact](/en/guide/ai-agents/tools/change-customer-contact) | Saves or updates phone, WhatsApp, or email |
| [Update Customer Address](/en/guide/ai-agents/tools/change-customer-address) | Saves an address (text or GPS) on the customer record |
| [Query API](/en/guide/ai-agents/tools/query-api) | Queries or runs operations on external APIs with allowlisted routes |
| [Research information on websites](/en/guide/ai-agents/tools/research-websites) | Reads web pages (Firecrawl or HTML) to answer with up-to-date content |
| [Transfer to Team](/en/guide/ai-agents/tools/transfer-to-team) | Hands the conversation off to a human team |
| [Schedule](/en/guide/ai-agents/tools/schedule) | Creates appointments from the conversation |
| [Confirm Appointment](/en/guide/ai-agents/tools/confirm-schedule) | Confirms attendance for existing appointments |
| [Change Funnel](/en/guide/ai-agents/tools/change-funnel) | Moves the customer between sales funnel stages |
| [When Unable to Answer](/en/guide/ai-agents/tools/unknown-response) | Defines what to do when the AI cannot find an answer |
| [Change Return Date](/en/guide/ai-agents/tools/change-return-date) | Reschedules the customer’s automatic follow-up |
