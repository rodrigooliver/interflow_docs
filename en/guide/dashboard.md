# Dashboard

Customize your home screen with the most relevant widgets for your daily work.

## Overview

Interflow's Dashboard is fully customizable for each user. You can choose, reorder, and remove widgets from available modules — CRM, financial, appointments, tasks, and conversations — keeping the most important information right at the entrance to the system.

::: tip 💡 Access
Access the Dashboard via the **home icon** in the left menu.
:::

## Available Widgets

| Widget | Description |
|--------|-------------|
| **Personal Statistics** | Cards with summary of active conversations, tasks, and appointments |
| **Recent Conversations** | Latest conversations assigned to the user |
| **Tasks — List** | Pending and in-progress tasks |
| **Tasks — Kanban** | Full view in columns by stage |
| **Appointments** | Upcoming scheduled appointments |
| **CRM Funnel** | Summary of customers by sales funnel stage |
| **Financial Summary** | Cash balances and general financial overview |

## Customizing the Dashboard

1. Access the **Dashboard** from the left menu
2. Click the **"Customize"** button in the top-right corner of the welcome banner
3. In the modal, **drag** widgets to reorder or use the icons to:
   - **Change size** (↔ between half-width and full-width)
   - **Remove** the widget (🗑)
4. In the **"Add widgets"** section, click **+** to include new modules
5. Click **"Save configuration"** — the layout is saved for future sessions

::: info 💾 Sync
The configuration is saved in the database and synchronized across all your devices.
:::

## Widget Sizes

Each widget can be displayed in two sizes:

| Size | Description |
|------|-------------|
| **Half-width** | Two widgets side by side on the same line |
| **Full-width** | Widget occupies all available width |

Toggle size by clicking the ↔ icon during customization.

## View as User (Administrators)

Administrators and owners can check how the dashboard looks for regular users:

1. In the admin dashboard, click **"View as user"**
2. View and configure the dashboard in regular user mode
3. The selected mode is saved and automatically restored on each access
