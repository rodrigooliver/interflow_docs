# API Keys

Generate keys to call the [Interflow API](/en/api/).

::: tip Access
**Settings → API Keys**
:::

1. **New key** → name it → copy once → store securely
2. Authenticate as described in [Authentication](/en/api/authentication)

::: danger
Secret keys are usually shown **only once**. Revoke and recreate if lost.
:::
