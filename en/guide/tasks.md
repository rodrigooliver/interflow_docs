# My Tasks

View and manage all tasks assigned to you, regardless of project, in one place.

## Overview

The **My Tasks** page brings together all your tasks in a unified view, without needing to navigate project by project. The project name appears next to each task title for easy context.

::: tip 💡 Access
Access via **Tasks → "All my tasks"** in the top bar.
:::

## Views

### List

Displays all tasks in a table format with:
- Title and source project
- Status, priority, and due date
- Assignee
- Grouping and sorting controls

### Calendar

Monthly view of tasks by due date.

## Groupings

Choose how to organize your tasks in the list view:

| Grouping | Description |
|----------|-------------|
| **Stage** | Traditional grouping by project flow (default) |
| **Project** | Groups by source project (available in My Tasks) |
| **Due date** | Smart groups: Overdue, Yesterday, Today, Tomorrow, This week, Next week, Future, No date |
| **Assignee** | One group per team member |
| **Status** | Pending, In progress, Completed, Cancelled |
| **Priority** | High, Medium, Low |

::: info 💾 Saved preference
Grouping choice is automatically saved per organization.
:::

## Sorting

Define the order of tasks within each group:

- **Default** — priority + due date
- **Due date (soonest)**
- **Due date (furthest)**
- **Priority**
- **Title** (A–Z or Z–A)
- **Creation date**
- **Status**

## Quick Edit in List

Change status, priority, and due date directly in the list, without opening the task:

1. Click on the **status**, **priority**, or **due date** of any task
2. Choose the new value from the menu
3. For due date, use shortcuts: **Today**, **Tomorrow**, **In 1 week**, custom date, or remove date

::: tip 💡 Notifications preserved
Status changes (start, complete, cancel) notify project members and record history, exactly like regular actions.
:::

## Filters

- **Search** — search by title
- **Overdue** — show only tasks with passed deadlines
- **Archived** — include archived tasks
- **Priority** — filter by priority level
- **Status** — filter by status

## Navigation

| Action | How to |
|--------|--------|
| Go to My Tasks | **Tasks → "All my tasks"** |
| Return to project view | Click **"View by project"** |
| Reload tasks | Click **"Refresh"** |

## Creating Tasks from Messages

You can create tasks directly from chat conversations:

1. In a conversation's message area, select one or more messages
2. In the bottom bar, click **"Create Task"**
3. Fill in the details (title, assignee, deadline)
4. The selected messages are linked to the task as context
