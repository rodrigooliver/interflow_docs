# Minhas Tarefas

Visualize e gerencie todas as tarefas atribuídas a você, independente do projeto, em um único lugar.

## Visão Geral

A página **Minhas Tarefas** reúne todas as suas tarefas em uma visão unificada, sem precisar navegar projeto por projeto. O nome do projeto de origem aparece ao lado do título de cada tarefa para facilitar o contexto.

::: tip 💡 Acesso
Acesse em **Tarefas → "Todas as minhas tarefas"** na barra superior.
:::

## Visualizações

### Lista

Exibe todas as tarefas em formato de tabela com:
- Título e projeto de origem
- Status, prioridade e data de vencimento
- Responsável
- Controles de agrupamento e ordenação

### Calendário

Visualização mensal das tarefas por data de vencimento.

## Agrupamentos

Escolha como organizar suas tarefas na visualização de lista:

| Agrupamento | Descrição |
|-------------|-----------|
| **Etapa** | Agrupamento tradicional pelo fluxo do projeto (padrão) |
| **Projeto** | Agrupa pelo projeto de origem (disponível em Minhas Tarefas) |
| **Data de vencimento** | Grupos: Vencidas, Ontem, Hoje, Amanhã, Esta semana, Próxima semana, Futuras, Sem data |
| **Responsável** | Um grupo por membro da equipe |
| **Status** | Pendente, Em andamento, Concluído, Cancelado |
| **Prioridade** | Alta, Média, Baixa |

::: info 💾 Preferência salva
A escolha de agrupamento é salva automaticamente por organização.
:::

## Ordenação

Defina a ordem das tarefas dentro de cada grupo:

- **Padrão** — prioridade + vencimento
- **Vencimento mais próximo**
- **Vencimento mais distante**
- **Prioridade**
- **Título** (A–Z ou Z–A)
- **Data de criação**
- **Status**

## Edição Rápida na Lista

Altere status, prioridade e vencimento diretamente na lista, sem abrir a tarefa:

1. Clique sobre o **status**, a **prioridade** ou o **vencimento** de qualquer tarefa
2. Escolha o novo valor no menu
3. Para vencimento, use os atalhos: **Hoje**, **Amanhã**, **Em 1 semana**, data livre ou remover data

::: tip 💡 Notificações preservadas
As mudanças de status (iniciar, concluir, cancelar) notificam os membros do projeto e registram o histórico, exatamente como as ações normais.
:::

## Filtros

- **Pesquisa** — buscar por título
- **Vencidas** — mostrar apenas tarefas com prazo ultrapassado
- **Arquivadas** — incluir tarefas arquivadas
- **Prioridade** — filtrar por nível de prioridade
- **Status** — filtrar por status

## Navegação

| Ação | Como fazer |
|------|------------|
| Ir para Minhas Tarefas | **Tarefas → "Todas as minhas tarefas"** |
| Voltar à visão por projeto | Clique em **"Ver por projeto"** |
| Recarregar tarefas | Clique em **"Atualizar"** |

## Criando Tarefas a partir de Mensagens

É possível criar tarefas diretamente de conversas no chat:

1. Na área de mensagens de uma conversa, selecione uma ou mais mensagens
2. Na barra inferior, clique em **"Criar Tarefa"**
3. Preencha os detalhes (título, responsável, prazo)
4. As mensagens ficam vinculadas à tarefa como contexto
