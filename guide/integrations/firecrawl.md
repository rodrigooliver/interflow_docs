# Integração Firecrawl

Use o Firecrawl para extrair conteúdo de páginas web em automações e agentes de IA.

## Configurar

1. Crie uma conta e API key no Firecrawl
2. No Interflow: **Configurações → Integrações**
3. Localize **Firecrawl**
4. Cole a chave e salve

## Uso típico

- Enriquecer contexto de [agentes IA](/guide/ai-agents/)
- Buscar conteúdo público para respostas mais precisas

::: tip
Combine com bases de conhecimento internas do agente para equilibrar dados da empresa e conteúdo da web.
:::
