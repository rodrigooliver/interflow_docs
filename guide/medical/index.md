# Módulo Médico (EMR)

Prontuário eletrônico ligado ao CRM: pacientes, consultas, receitas, atestados e templates clínicos.

::: tip Acesso
Menu → **Prontuário / Médico**. Requer módulo `medical` habilitado.
:::

::: info Paciente = Cliente
Cada paciente é um **cliente** do CRM. O histórico clínico fica unificado na ficha do paciente.
:::

## Visão geral

| Área | Função |
|------|--------|
| **Consultas** | Atendimentos clínicos |
| **Prontuários** | Registros médicos |
| **Receitas** | Prescrições |
| **Atestados** | Certificados / atestados |
| **Templates** | Modelos de documentos clínicos |
| **Pacientes** | Lista e ficha unificada |

## Ficha do paciente

1. Abra **Médico → Pacientes**
2. Selecione o paciente
3. Use as abas: agendamentos, prontuários, receitas, atestados e anexos

## Fluxo típico

1. Localize ou cadastre o cliente/paciente
2. Crie uma **consulta**
3. Registre o **prontuário** da consulta
4. Emita **receita** e/ou **atestado** a partir de templates, se necessário
5. Anexe arquivos à ficha quando precisar

## Templates médicos

Em **Médico → Templates**, cadastre modelos reutilizáveis para documentos clínicos (estrutura semelhante aos [modelos de documentos](/guide/documents/) da plataforma).
