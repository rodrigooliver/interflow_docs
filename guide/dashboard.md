# Dashboard

Personalize sua tela inicial com os widgets mais relevantes para sua rotina de trabalho.

## Visão Geral

O Dashboard da Interflow é totalmente personalizável por cada usuário. Você pode escolher, reordenar e remover widgets dos módulos disponíveis — CRM, financeiro, agendamentos, tarefas e conversas — deixando as informações mais importantes logo na entrada do sistema.

::: tip 💡 Acesso
Acesse o Dashboard pelo **ícone inicial** no menu lateral.
:::

## Widgets Disponíveis

| Widget | Descrição |
|--------|-----------|
| **Estatísticas Pessoais** | Cards com resumo de conversas ativas, tarefas e agendamentos |
| **Conversas Recentes** | Últimas conversas atribuídas ao usuário |
| **Tarefas — Lista** | Tarefas pendentes e em andamento |
| **Tarefas — Kanban** | Visualização completa em colunas por estágio |
| **Agendamentos** | Próximos compromissos agendados |
| **Funil CRM** | Resumo de clientes por etapa do funil de vendas |
| **Resumo Financeiro** | Saldo dos caixas e visão financeira geral |

## Personalizando o Dashboard

1. Acesse o **Dashboard** pelo menu lateral
2. Clique no botão **"Personalizar"** no canto superior direito do banner de boas-vindas
3. No modal, **arraste** os widgets para reordenar ou use os ícones para:
   - **Alterar tamanho** (↔ entre meia largura e largura total)
   - **Remover** o widget (🗑)
4. Na seção **"Adicionar widgets"**, clique em **+** para incluir novos módulos
5. Clique em **"Salvar configuração"** — o layout fica salvo para próximas sessões

::: info 💾 Sincronização
A configuração é salva no banco de dados e sincronizada entre todos os seus dispositivos.
:::

## Tamanho dos Widgets

Cada widget pode ser exibido em dois tamanhos:

| Tamanho | Descrição |
|---------|-----------|
| **Meia largura** | Dois widgets lado a lado na mesma linha |
| **Largura total** | Widget ocupa toda a largura disponível |

Alterne o tamanho clicando no ícone ↔ durante a personalização.

## Visualização como Usuário (Administradores)

Administradores e proprietários podem verificar como o dashboard aparece para usuários comuns:

1. No dashboard admin, clique em **"Visualizar como usuário"**
2. Veja e configure o dashboard no modo de usuário normal
3. O modo selecionado é salvo e restaurado automaticamente a cada acesso

## Resetar Configuração

Para voltar ao layout padrão:

1. Clique em **"Personalizar"**
2. Remova os widgets indesejados ou reordene conforme necessário
3. Salve
